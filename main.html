MeloTab v1.5.0

Productivity dashboard for your Chrome New Tab

Short Description

Custom new tab with search, tasks, weather, calendar, shortcuts, Gmail badge, AI launcher, and private file sharing.

Key Features

🔍 Multi-engine Search
📅 Google Calendar Integration
☁️ Live Weather Forecast
📧 Gmail Unread Badge
🤖 AI Tool Launcher
📦 MeloDrop P2P Sharing
✅ Persistent Todo List
🎨 Auto Light/Dark Themes

Privacy & Security

  • No Tracking: No analytics or tracking scripts.
  • Local First: Data stays in your Chrome storage.
  • No Ads: Clean, distraction-free interface.
  • Direct P2P: Files are shared browser-to-browser via WebRTC.

Permissions Justification

storage: Saving settings and your local todo list.

geolocation: Accurate weather updates (optional).

host permissions: Accessing weather APIs, search suggestions, and your Calendar/Gmail feeds directly.
